# Full Stack E‑Com (MERN)

Monorepo with three apps:
- `Backend/` — Express API + MongoDB
- `Fronted/` — Customer storefront (React + Vite + Tailwind)
- `Admin/` — Admin panel (React + Vite)

## Prerequisites
- Node.js 18+
- MongoDB

## Quick Start
```bash
cd Backend
npm install
npm run dev
```

```bash
cd ../Fronted
npm install
npm run dev
```

```bash
cd ../Admin
npm install
npm run dev
```
